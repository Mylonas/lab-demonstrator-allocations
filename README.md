# lab-demonstrator-allocations

An undergraduate thesis project that simplifies the lab demonstrator selection process through a web application built in Microsoft Visual Studio.

**Live thesis:** [https://mylonas.github.io/lab-demonstrator-allocations](https://mylonas.github.io/lab-demonstrator-allocations)

## Overview

The project addresses the manual, error-prone process of allocating lab demonstrators to sessions. The web application automates demonstrator selection by matching availability, module knowledge, and workload constraints.

## Background

Submitted as part of an undergraduate degree programme. The system was developed in Microsoft Visual Studio and uses a web-based interface to streamline the allocation workflow for lab coordinators.
